# 提示卡分表工具

将总表中的每行数据拆分为独立 sheet，方便阅读。

## 用法

1. 从 GitHub Actions 下载最新编译的 `提示卡分表工具.exe`
2. 把要处理的 `.xlsx` 文件**拖拽**到 exe 图标上
3. 自动生成 `原文件名（分表）.xlsx`

## 功能

- 每个车组生成一个 sheet，以「机械师+车次」命名
- 新建「查询页」，带超链接可跳转到对应 sheet
- 原文件不变，生成新文件
- 支持文件拖拽

## 数据结构要求

| 第1行 | 导入说明（忽略） |
| 第2行 | 表头 |
| 第3行起 | 数据行 |

| A列 | B列 | C列 | D列 | E列 |
|:---|:---|:---|:---|:---|
| 交路名称 | 日期 | **车组号** | **车次** | **机械师** |

C列(车组号)不能为空，D列(车次)+E列(机械师)用于命名 sheet。
